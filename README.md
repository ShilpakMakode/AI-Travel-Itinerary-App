AI Travel Itinerary App

An intelligent travel planning application that creates personalized, day-wise itineraries using LLMs.

The app takes user inputs such as destination, budget, trip duration, and interests, then generates structured travel plans with relevant activity suggestions and scheduling guidance.

Features

Personalized itinerary generation
Day-by-day trip planning

Preference-based recommendations

Modular architecture for easy extension

Designed for future API integrations (hotels, transport, weather)
